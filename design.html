기본아이콘

홈페이지 주요 기능에 대한 아이콘을 설정합니다. 설정 가능한 기능은 다음과 같습니다.

  • 장바구니
  • 로그인
  • 로그아웃
  • 마이 페이지
  • 모바일 메뉴 바
  • 포토구매평
  • 구매평 별점
  • QNA 비밀글

각 기능에 대해 개별 아이콘 설정이 가능하며 아이콘은 지속적으로 업데이트될 예정입니다. 현재 사용가능한 아이콘 종류는 다음과 같습니다. (v1.5)

색상

홈페이지에서 기본적으로 사용할 색상을 선택합니다. 설정 가능한 항목은 다음과 같습니다.

  • 가격 글자 색상

    상품 리스트, 상품 상세, 장바구니, 주문정보 입력 등의 화면에서 일괄 적용 됩니다.

    ex) 팬시유 색상 ●(#6930c3) 적용

    [상품 리스트]

    [상품 상세]

    [장바구니]

    [주문정보 입력]

  • 버튼 글자 색상

    로그인 / 구매하기 / 결제하기 등 주요 버튼, 상품 상세 페이지 내 SALE 라벨 / 구매평·문의 개수, 체크박스, 페이지 버튼 등의 버튼 형식의 글자 색상에 일괄 적용 됩니다.

    ex) 팬시유 색상 ○(#ffffff) 적용

    [로그인]

    [SALE라벨, 구매하기]

    [글 개수, 페이지]

    [체크박스]

  • 버튼 배경 색상

    로그인 / 구매하기 / 결제하기 등 주요 버튼, 상품 상세 페이지 내 SALE 라벨 / 구매평·문의 개수, 체크박스, 페이지 버튼 등의 버튼 형식의 배경 색상에 일괄 적용 됩니다.

    ex) 팬시유 색상 ●(#6930c3) 적용

    [로그인]

    [SALE라벨, 구매하기]

    [글 개수, 페이지]

    [체크박스]

  • 구매평 별점 색상

    상품 상세 상단 구매평, 일반 구매평, 구매평 작성 모달 등 구매평 별점 색상에 일괄 적용 됩니다. 별점에 따라 채워지지 않은 영역(5점이 아닌 경우)은 불투명하게 표시됩니다.

    ex) 팬시유 색상 ●(#6930c3) 적용

    [상품 상세 상단 구매평]

    [상품 상세 일반 구매평]

    [구매평 작성 모달]

헤더 설정

홈페이지의 헤더 설정을 할 수 있습니다. 정의되어 있는 헤더 타입을 선택하고, 헤더 내부 동작과 색상, 순서등의 기본 레이아웃 설정이 가능합니다. 데스크탑의 경우 헤더 타입에 따라 홈페이지 전체 페이지 레이아웃이 변경될 수 있으며, 모바일의 경우 기본 상단 헤더 스타일이 유지 + 모바일 사이드메뉴 별도 설정이 가능합니다. 총 5개의 메뉴로 구성되어 있으며, 각 메뉴는 아래 그림과 함께 설명을 참고해 주세요.

  • 기본 레이아웃 : 타입 선택, 헤더 높이, 기본 색상, 내용 위치, 오버랩 설정으로 구성됩니다.

    • 타입 선택 : 원하는 헤더 타입을 선택합니다.

    • 헤더 높이 : 로고 높이와 상하 여백을 입력합니다. 두 값의 합이 전체 헤더 높이가 됩니다. 데스크탑, 모바일 각각 입력합니다.

    • 기본 색상 : 헤더의 기본 배경과 텍스트 색상을 선택합니다. 항목 클릭 후 팔레트에서 선택하거나 색상코드를 직접 입력합니다.

    • 내용 위치 : 로고 / 페이지 / 기능의 위치를 선택합니다. 데스크탑, 모바일 각각 설정하고, 데스크탑의 경우 헤더 타입에 따라 선택 항목이 다르게 나타납니다.

      [헤더 타입 A]

      [헤더 타입 B]

    • 오버랩 적용 : 데스크탑 / 모바일의 헤더 오버랩 여부를 선택합니다. 활성화 한 경우 오버랩 텍스트 색상 설정도 가능합니다.

    • 데스크탑 와이드 레이아웃 적용 : 데스크탑 헤더의 와이드 레이아웃 적용 여부를 선택합니다.

  • 기능 : 홈페이지 기본 기능은 카트, 로그인, 로그아웃, 마이페이지로 구성됩니다. 각 기능에 대해 이름과 표시 방식을 설정합니다.

    • 이름 : 원하는 이름을 입력합니다.

    • 표시 방식 : 원하는 표시방식을 선택합니다. 데스크탑, 모바일 별도 설정이 가능합니다.

  • 페이지 : 홈페이지에서 추가로 사용할 페이지를 설정합니다.

    • 이름 : 원하는 이름을 입력합니다.

    • 타입 : 페이지의 타입을 선택합니다. 타입에 따라 경로 입력이 결정됩니다.

      [타입 - 상품리스트]

      [타입 - 커스텀]

      [타입 - 외부링크]

    • 경로 : 페이지의 경로를 입력합니다. 타입이 '커스텀' 일 경우 내부 페이지 경로를 입력하고, '외부링크' 일 경우 해당 URL을 입력합니다.

  • 스티키 : 헤더의 스티키 동작여부를 선택합니다. 데스크탑 / 모바일 별도 설정이 가능합니다.

    • 기본 : 상품 상세 페이지를 제외한 모든 페이지에 적용될 동작 방식을 선택합니다.

    • 상품 상세 페이지 : 상품 상세 페이지에 적용될 동작 방식을 선택합니다.

    • 색상 : 스티키가 활성화 되었을 때, 나타날 배경과 텍스트 색상을 설정합니다.

  • 모바일 사이드메뉴 : 모바일 사이드메뉴 디자인을 설정합니다.

    • 타입 선택 : 원하는 모바일 사이드메뉴 타입을 선택합니다.

    • 기본 색상 : 모바일 사이드메뉴의 배경과 텍스트 색상을 선택합니다.

기본 레이아웃

헤더 기본 디자인 설정 메뉴 입니다. 타입 선택, 헤더 높이, 기본 색상 설정, 내용 위치 설정, 기타 설정으로 구성됩니다.

  • 타입 선택

    헤더 타입을 선택 할 수 있습니다. 데스크탑 레이아웃에 영향을 주며, 현재 사용 가능한 타입은 A(세로형), B(가로형) 입니다. A타입은 상단 헤더 + 아래 컨텐츠 구조의 세로 형식 레이아웃 이고, B타입은 좌/우 헤더 + 옆 컨텐츠 구조의 가로 형식 레이아웃 입니다. 모바일에서는 상단 헤더 레이아웃이 유지됩니다.

    ex) A타입(내음) 적용

    [메인 페이지]

    [상품 상세]

    ex) B타입(오소묘) 적용

    [메인 페이지]

    [상품 상세]

  • 헤더 높이

    헤더 높이를 px단위로 입력 합니다. 로고 높이와 상하여백을 각각 입력하고, 두 크기의 합이 전체 헤더 높이가 됩니다. 로고의 가로 사이즈는 입력한 높이에 따라 비율이 자동계산 되어 표시됩니다.

    ex) 내음 데스크탑(로고 : 45px, 상하여백 : 35px)

    [전체 헤더 높이 : 80px]

  • 기본 색상 설정

    헤더의 배경과 텍스트 색상을 설정합니다. 헤더 상태가 스티키와 오버랩일 경우를 제외한 기본 헤더 상태에서만 동작합니다. 로고이미지를 제외한 페이지/기능/메뉴바 등 헤더에 포함된 모든 텍스트 및 아이콘에 적용됩니다.

    ex) 센르 텍스트 ●(#000000) , 배경 ●(#fffbf0) 적용

    [텍스트]

    [기능 아이콘, 메뉴바]

  • 내용 위치 설정

    헤더 내부 내용의 위치를 설정합니다. 헤더 타입, 데스크탑 / 모바일 여부에 따라 설정 가능한 항목이 상이합니다. 각 항목은 활성화 상태일 때만 적용됩니다.

    데스크탑

    • 헤더 타입 A - 로고, 페이지, 기능

      로고, 페이지, 기능의 위치를 설정합니다. 왼쪽/가운데/오른쪽/숨김 중 한가지를 선택합니다.

      ex) 센르 로고 위치 설정

      [왼쪽]
